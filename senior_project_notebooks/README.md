# Senior-project exploration notebooks

Early-stage exploration notebooks from the underwater object-detection senior-project track (before the project pivoted to the SAR drone direction tracked in [sar-drone](https://github.com/AlghamdiMuath/sar-drone)).

- `cameraTest.ipynb` — initial camera capture & preview experiments.
- `spiderGraph_CDR.ipynb` — radar-chart visualizations for the Critical Design Review.
- `Underwater_Video_Processing_Workflow.svg` — diagram of the proposed underwater video processing pipeline.

Kept here for reference alongside the underwater-preprocessing pipeline in the parent repo.
